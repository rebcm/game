# Guia de Instalação

Este guia fornece as instruções necessárias para configurar e executar o jogo Rebeca em seu ambiente de desenvolvimento.

## Pré-requisitos

1. **Versão do Flutter**: Certifique-se de que você tem a versão `>=2.12.0 <3.0.0` do Flutter instalada.
2. **Dart SDK**: Verifique se a versão do Dart SDK é compatível com a versão do Flutter instalada.
3. **Configuração de Emulador**: Configure um emulador Android ou iOS para executar o aplicativo.
4. **Variáveis de Ambiente**: Configure as variáveis de ambiente necessárias para o projeto.

## Passos para Instalação

1. Clone o repositório do projeto.
2. Execute `flutter pub get` para instalar as dependências.
3. Configure as variáveis de ambiente de acordo com as necessidades do projeto.
4. Execute `flutter run` para iniciar o aplicativo no emulador configurado.

## Checklist de Verificação

- [ ] Versão do Flutter: `>=2.12.0 <3.0.0`
- [ ] Dart SDK compatível com a versão do Flutter
- [ ] Emulador configurado e funcionando
- [ ] Variáveis de ambiente configuradas corretamente
