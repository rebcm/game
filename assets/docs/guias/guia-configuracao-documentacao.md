# Guia de Configuração da Documentação

Este guia ajuda a configurar a documentação do projeto Rebeca.

## Passos

1. **Verifique o arquivo `README.md`**: Certifique-se de que o conteúdo está atualizado e correto.
2. **Execute o workflow de definição de hospedagem**: Vá para as Actions do repositório e execute o workflow "Definir Hospedagem da Documentação" com a opção desejada.

## Dicas

- Mantenha a documentação concisa e relevante.
- Utilize markdown para formatação.

## Checklist

- [ ] Conteúdo do README.md está atualizado.
- [ ] Workflow de hospedagem executado com sucesso.
