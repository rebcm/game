# Matriz de Velocidade vs Frame Rate

Esta matriz correlaciona a velocidade de deslocamento (m/s) com a velocidade de reprodução da animação para evitar o efeito de deslizamento.

| Velocidade (m/s) | Frame Rate |
|------------------|------------|
| 0.5              | 10         |
| 1.0              | 20         |
| 1.5              | 30         |
| 2.0              | 40         |
| 2.5              | 50         |
| 3.0              | 60         |

## Notas

* A velocidade de deslocamento é medida em metros por segundo.
* O frame rate é medido em quadros por segundo.
* A matriz foi gerada com base em testes de movimento e animação.

## Uso

Para usar essa matriz, basta encontrar a velocidade de deslocamento desejada e ajustar o frame rate da animação de acordo.
