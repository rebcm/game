# Mapeamento de Coeficientes de Fricção e Elasticidade

Este documento descreve os coeficientes de fricção e elasticidade utilizados no jogo para diferentes superfícies.

## Tabela de Materiais

| Material | Coeficiente de Fricção | Coeficiente de Elasticidade |
| --- | --- | --- |
| Bloco de Terra | 0.5 | 0.2 |
| Bloco de Pedra | 0.7 | 0.1 |
| Bloco de Madeira | 0.4 | 0.3 |
| Rebeca | 0.3 | 0.4 |

## Explicação dos Coeficientes

*   Coeficiente de Fricção: Define a resistência ao movimento entre duas superfícies.
*   Coeficiente de Elasticidade: Define a capacidade de uma superfície de recuperar sua forma original após uma colisão.

## Implementação

Os coeficientes definidos nesta tabela serão utilizados na implementação do comportamento do veículo no jogo, garantindo uma experiência de jogo realista e consistente.
