# Checklist de Validação de Conteúdo

## Descrição para Lojas de Aplicativos

- [ ] Limite de caracteres respeitado:
  - Título: 30 caracteres
  - Descrição curta: 80 caracteres
  - Descrição longa: 4000 caracteres
- [ ] Palavras-chave incluídas para SEO:
  - "Rebeca"
  - "Voxel"
  - "Criativo"
  - "Blocos"
- [ ] Ortografia e gramática verificadas

## Conteúdo do Jogo

- [ ] Descrição do modo criativo presente
- [ ] Menção às funcionalidades principais:
  - Construção
  - Exploração
- [ ] Ausência de referências a funcionalidades não implementadas

## SEO e Descoberta

- [ ] Palavras-chave relevantes incluídas na descrição
- [ ] Título e descrição curta atraentes e informativos

## Internacionalização

- [ ] Traduções disponíveis para as descrições e conteúdo
- [ ] Arquivos de tradução atualizados
