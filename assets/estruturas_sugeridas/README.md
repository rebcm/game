# Templates de Estruturas Sugeridas

Este diretório contém exemplos visuais e esquemáticos de estruturas sugeridas para o jogo, visando facilitar a compreensão e inspiração dos jogadores.

## Estruturas Disponíveis

- casa_basica.png
- torre_alta.png
- ponte_simples.png

## Adicionando Novas Estruturas

1. Crie sua imagem representativa da estrutura no formato PNG.
2. Adicione a imagem a este diretório.
3. Atualize a lista acima em `README.md`.

## Validação

As estruturas são validadas através do script `./.github/scripts/docs_validation/estruturas_sugeridas/validate_estruturas_sugeridas.sh`.
