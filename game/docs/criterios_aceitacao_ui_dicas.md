# Critérios de Aceitação para UI de Dicas

## Introdução

Este documento define os critérios de aceitação para a UI de dicas do jogo.

## Critérios de Aceitação

1. A UI de dicas deve ser exibida quando o jogador realizar uma ação específica.
2. A UI de dicas deve conter informações relevantes sobre o jogo.
3. A UI de dicas deve ser fechada quando o jogador clicar fora dela.

## Métricas de Sucesso

1. A UI de dicas deve ser exibida corretamente em diferentes dispositivos.
2. A UI de dicas deve conter as informações corretas.

## Comportamento Esperado

1. A UI de dicas deve ser exibida quando o jogador realizar uma ação específica.
2. A UI de dicas deve ser fechada quando o jogador clicar fora dela.
