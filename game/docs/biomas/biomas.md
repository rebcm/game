# Biomas do Jogo

O jogo atualmente possui um único bioma, que é o mundo criativo padrão. Abaixo está uma descrição detalhada do bioma disponível.

## Bioma Padrão

O bioma padrão é um mundo criativo onde os jogadores têm liberdade para construir e explorar. Este bioma é caracterizado por:

- **Terreno**: O terreno é gerado proceduralmente, oferecendo uma variedade de paisagens para os jogadores explorarem e construírem.
- **Blocos**: Os blocos disponíveis permitem uma ampla gama de construções, desde estruturas simples até criações complexas.
- **Iluminação**: A iluminação é dinâmica, permitindo que os jogadores criem ambientes com diferentes condições de luz.

### Características

- **Flexibilidade**: Os jogadores têm a liberdade de construir qualquer coisa que imaginarem.
- **Exploração**: O terreno gerado proceduralmente oferece uma experiência de exploração única a cada partida.
- **Criatividade**: A variedade de blocos e a iluminação dinâmica incentivam a criatividade e a experimentação.

### Uso

Para aproveitar ao máximo o bioma padrão, os jogadores podem:
1. Explorar o terreno gerado para encontrar locais interessantes para construir.
2. Utilizar a variedade de blocos disponíveis para criar estruturas e artefatos.
3. Experimentar com a iluminação para criar ambientes únicos.
