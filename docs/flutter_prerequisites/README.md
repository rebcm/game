# Pré-requisitos para Flutter

Este documento lista as versões específicas do Flutter SDK, Dart e dependências de sistema necessárias para o projeto.

## Versão do Flutter
Flutter >= 3.0.0

## Versão do Dart
Dart >= 2.17.0 < 3.0.0

## Dependências de Sistema
- Lista de dependências instaladas no sistema

## Guia de Configuração
1. Instale o Flutter seguindo as instruções em [Flutter Documentation](https://docs.flutter.dev/get-started/install).
2. Verifique a versão do Dart usando `dart --version`.
3. Execute `flutter doctor` para verificar as dependências de sistema necessárias.
