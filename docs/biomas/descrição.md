# Biomas do Jogo

O jogo atualmente apresenta um único bioma, que é um ambiente criativo onde os jogadores podem construir livremente.

## Características do Bioma

- **Terreno**: O terreno é gerado proceduralmente, oferecendo uma superfície variada para construção.
- **Blocos**: Os jogadores têm acesso a uma variedade de blocos voxel para construir estruturas, terrenos e arte.
- **Iluminação**: A iluminação é simples e uniforme, permitindo que os jogadores foquem na criatividade sem restrições de iluminação complexa.

## Detalhes Adicionais

- O bioma não possui elementos de sobrevivência, como recursos limitados, inimigos ou condições climáticas.
- A ênfase está na liberdade criativa, permitindo que os jogadores expressem suas ideias sem restrições.

## Futuras Atualizações

Embora o jogo atualmente tenha um único bioma, futuras atualizações podem incluir novos blocos ou características que expandam as possibilidades criativas dentro do bioma existente.
