# Biomas do Jogo

O jogo atualmente apresenta um único bioma, que é o mundo criativo padrão. Este bioma é caracterizado por uma paisagem voxel simples, onde os jogadores podem construir livremente sem restrições de sobrevivência ou ameaças.

## Características do Bioma

- **Terreno**: O terreno é gerado proceduralmente, oferecendo uma superfície plana e infinita para construção.
- **Blocos**: Os blocos disponíveis são variados, permitindo a construção de estruturas diversas, desde simples casas até complexas máquinas voxel.
- **Iluminação**: A iluminação é uniforme, permitindo que os jogadores construam sem se preocupar com a iluminação natural.

## Detalhes Adicionais

- **Ausência de NPCs e Monstros**: O jogo não inclui NPCs ou monstros, focando exclusivamente na experiência criativa do jogador.
- **Modo Criativo Puro**: O jogo é projetado para ser um modo criativo puro, sem elementos de sobrevivência ou perigo.
