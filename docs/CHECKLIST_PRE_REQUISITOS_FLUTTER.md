# Checklist de Pré-requisitos Flutter

## Versões Necessárias

- Flutter SDK: >=3.0.0
- Dart SDK: >=2.17.0 <3.0.0

## Dependências de Sistema

- Liste as dependências de sistema necessárias para o projeto.

## Passos para Validação

1. Execute `flutter --version` para verificar a versão do Flutter SDK.
2. Execute `dart --version` para verificar a versão do Dart SDK.
3. Execute `dart pub deps` para verificar as dependências do projeto.
4. Execute `dart analyze` para verificar a integridade do código.
