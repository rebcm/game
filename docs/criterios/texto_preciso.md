# Critérios para Texto Preciso

Para ser considerado 'preciso para o jogador', o texto deve atender aos seguintes critérios mínimos:

1. **Clareza**: O texto deve ser claro e fácil de entender.
2. **Concisão**: O texto deve ser conciso e direto.
3. **Relevância**: O texto deve ser relevante para o contexto do jogo.

Esses critérios garantem que o texto seja eficaz em comunicar informações ao jogador.
