# Critérios de Aceitação para Deploy Cloudflare

## Introdução

Este documento define os critérios de aceitação para o deploy do jogo Rebeca na Cloudflare Pages.

## Critérios de Aceitação

1. **Build web gerado com sucesso**: O build web do jogo deve ser gerado sem erros.
2. **Upload concluído via Wrangler**: O upload do build web para a Cloudflare Pages deve ser concluído com sucesso.
3. **URL de preview acessível**: A URL de preview gerada pela Cloudflare Pages deve ser acessível e exibir o jogo corretamente.

## Validação

A validação dos critérios de aceitação será realizada automaticamente através de um workflow do GitHub Actions.
