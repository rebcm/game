# Critérios de Aceitação para UI de Dicas

## Métricas de Sucesso
- A UI de Dicas é exibida corretamente quando o usuário passa o mouse sobre um bloco.
- A UI de Dicas é fechada quando o usuário clica fora da dica.

## Comportamentos Esperados
- A UI de Dicas deve ser exibida próxima ao bloco selecionado.
- A UI de Dicas deve conter informações relevantes sobre o bloco.

## Gatilhos de Exibição
- A UI de Dicas é exibida quando o usuário passa o mouse sobre um bloco.
- A UI de Dicas é fechada quando o usuário clica fora da dica.

## Persistência
- A UI de Dicas não persiste após o usuário fechar a dica.
