# Fluxo de Aprovação de Conteúdo

Este documento descreve o fluxo de aprovação de conteúdo para as dicas de construção.

## Processo de Aprovação

1. As dicas são criadas e armazenadas no arquivo `dicas.json`.
2. As dicas são extraídas e validadas pelo script `validate_fluxo_aprovacao_conteudo.sh`.
3. Se as dicas forem aprovadas, elas são publicadas na interface do jogo.
