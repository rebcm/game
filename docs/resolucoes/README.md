# Matriz de Resoluções de Tela

Este documento lista as resoluções de tela alvo para garantir a cobertura dos testes de renderização do jogo.

## Dispositivos Alvo

| Dispositivo | Resolução |
| --- | --- |
| iPhone SE | 640x1136 |
| Pixel 7 | 1080x2244 |
| iPad Air | 2224x1668 |
| Samsung Galaxy Tab S8 | 2220x1080 |

## Justificativa

A escolha desses dispositivos visa cobrir uma variedade de resoluções e proporções de tela comuns no mercado, garantindo que o jogo seja testado em diferentes configurações.

## Testes de Renderização

Para garantir que o jogo seja renderizado corretamente em diferentes resoluções, serão realizados testes de renderização nos dispositivos listados acima.
