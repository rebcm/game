# Matriz de Resoluções de Tela

Este documento define as resoluções de tela alvo para garantir a cobertura dos testes de renderização do jogo.

## Dispositivos Alvo

| Dispositivo | Resolução |
|-------------|-----------|
| iPhone SE   | 640x1136  |
| Pixel 7     | 1080x2400 |
| Tablet Genérico | 1280x800 |

## Justificativa

A escolha dessas resoluções visa cobrir uma gama de dispositivos móveis e tablets, garantindo que o jogo seja testado em diferentes tamanhos e densidades de tela.

## Testes de Renderização

Os testes de renderização devem ser executados em cada uma dessas resoluções para garantir que o jogo seja renderizado corretamente.
