# Matriz de Resoluções de Tela

Para garantir a cobertura adequada dos testes de renderização, definimos uma matriz de resoluções de tela alvo.

## Dispositivos Alvo

* iPhone SE (2ª geração) - 750x1334
* Pixel 7 - 1080x2400
* iPad (7ª geração) - 1080x2224 (portrait) e 2224x1080 (landscape)
* Samsung Galaxy Tab S8 - 1440x2560

## Justificativa

A escolha desses dispositivos visa cobrir uma variedade de tamanhos de tela e densidades de pixels, garantindo que o jogo seja testado em diferentes configurações comuns.

## Implementação

Para implementar a matriz de resoluções, atualize os testes de renderização para incluir as resoluções listadas acima.
