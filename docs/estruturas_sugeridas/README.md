# Estruturas Sugeridas

Este diretório contém exemplos de estruturas que podem ser construídas no jogo, 
servindo como inspiração ou dicas para os jogadores.

## Critérios para Adição de Estruturas

- Devem ser criativas e variadas.
- Devem seguir as regras do jogo (modo criativo, sem NPCs ou monstros, etc.).
- Devem ser úteis para os jogadores, oferecendo ideias interessantes.

## Processo de Validação

As estruturas sugeridas são validadas através do script `validate_estruturas_sugeridas.sh` 
localizado em `.github/scripts/docs_validation`.
