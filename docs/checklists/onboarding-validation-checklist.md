# Checklist de Validação de Onboarding

## Pré-requisitos
- Flutter instalado
- Dart instalado
- Configuração de ambiente de desenvolvimento

## Passos para Validação
1. Clonar o repositório
2. Executar `flutter pub get`
3. Executar `flutter run`
4. Verificar se o jogo inicia corretamente
5. Testar funcionalidades básicas (construção, movimentação)

## Critérios de Aceitação
- O jogo inicia sem erros
- Todas as funcionalidades básicas funcionam corretamente
