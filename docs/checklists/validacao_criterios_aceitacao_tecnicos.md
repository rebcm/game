# Critérios de Aceitação para Deploy Cloudflare

## Introdução

Este documento define os critérios de aceitação para o deploy do jogo Rebeca na Cloudflare Pages.

## Critérios

1. **Build web gerado com sucesso**: O build web do jogo deve ser gerado sem erros.
2. **Upload concluído via Wrangler**: O upload do build web para a Cloudflare Pages deve ser concluído com sucesso.
3. **URL de preview acessível**: A URL de preview da Cloudflare Pages deve ser acessível e exibir o jogo corretamente.

## Responsabilidades

- A equipe de desenvolvimento é responsável por garantir que o build web seja gerado corretamente.
- A equipe de DevOps é responsável por configurar e monitorar o upload via Wrangler.
- A equipe de QA é responsável por verificar a acessibilidade da URL de preview.
