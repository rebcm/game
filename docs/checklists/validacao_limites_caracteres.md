# Checklist de Validação de Limites de Caracteres

1. Verificar se os títulos e descrições para App Store estão dentro dos limites.
2. Verificar se os títulos e descrições para Play Store estão dentro dos limites.
3. Verificar se os títulos e descrições para Documentação Interna estão dentro dos limites.

## Critérios de Aceitação

- Todos os textos estão dentro dos limites definidos para cada canal.
- Nenhum texto é truncado ou cortado.
