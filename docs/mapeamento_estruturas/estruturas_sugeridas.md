# Mapeamento de Estruturas Sugeridas

Este documento visa listar as estruturas específicas do jogo que precisam de templates para garantir a cobertura de todos os cenários essenciais ao jogador.

## Estruturas Identificadas

1. **Blocos Básicos**
   - Descrição: Blocos fundamentais do jogo, como terra, pedra, areia, etc.
   - Templates Necessários: Sim

2. **Estruturas Naturais**
   - Descrição: Estruturas que ocorrem naturalmente no jogo, como árvores, rios, montanhas.
   - Templates Necessários: Sim

3. **Construções do Jogador**
   - Descrição: Estruturas criadas pelo jogador, como casas, pontes, etc.
   - Templates Necessários: Sim

4. **Estruturas Especiais**
   - Descrição: Estruturas com funcionalidades especiais, como portões, escadas, etc.
   - Templates Necessários: Sim

## Templates Necessários

Para cada estrutura identificada, os seguintes templates devem ser criados:
- Template de Bloco: Define a aparência e comportamento do bloco.
- Template de Estrutura: Define como os blocos se organizam para formar a estrutura.

## Implementação

A implementação dos templates será feita utilizando os recursos existentes no jogo, garantindo que sejam compatíveis com o modo criativo e não adicionem complexidade desnecessária.
