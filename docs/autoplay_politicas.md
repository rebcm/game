# Autoplay Policies for Major Browsers

This document outlines the autoplay policies for Chrome, Safari, and Firefox.

## Chrome Autoplay Policy

Chrome's autoplay policy allows autoplay for videos with muted audio. For videos with audio, autoplay is allowed if the user has interacted with the site.

## Safari Autoplay Policy

Safari's autoplay policy is more restrictive. Autoplay is allowed for videos with muted audio, but for videos with audio, autoplay is only allowed if the user has interacted with the site or if the video is served over a secure connection (HTTPS).

## Firefox Autoplay Policy

Firefox allows autoplay for videos with muted audio by default. For videos with audio, autoplay is blocked by default, but users can configure their preferences to allow autoplay for specific sites.
