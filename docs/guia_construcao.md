# Guia de Construção

## Versão do Flutter
Para construir o projeto, é necessário ter o Flutter na versão 3.0.0 ou superior instalado.

## Dependências
As dependências do projeto estão listadas no arquivo `pubspec.yaml`.

## Variáveis de Ambiente
As variáveis de ambiente necessárias para o projeto estão documentadas no arquivo `.env.example`.

## Comandos de Build
Para construir o projeto, execute o comando `flutter build` seguido das opções necessárias para a plataforma desejada.
