# Checklist de Validação de Documentação

## Tópicos Obrigatórios

- [ ] Versão do Flutter
- [ ] Dependências
- [ ] Variáveis de Ambiente
- [ ] Comandos de Build

## Guia de Uso

1. Verifique se todos os tópicos obrigatórios estão presentes no arquivo `dicas.md`.
2. Execute o script de validação `./.github/scripts/docs_validation/checklist_validacao_documentacao/validate_checklist_documentacao.sh` para verificar se os tópicos estão presentes.

## Manutenção

- Atualize este checklist sempre que necessário para refletir as mudanças no projeto.
