# Versão do Flutter SDK

## Introdução

Este documento fornece informações sobre a versão do Flutter SDK utilizada no projeto.

## Versão Atual

A versão atual do Flutter SDK é especificada no arquivo `pubspec.yaml`.

## Versões Mínimas Suportadas

- Android: API Level 21 (Android 5.0)
- iOS: iOS 12

## Histórico de Versões

- Versão 1.0.0: Flutter SDK 3.0.0
