# Mapeamento de Versão Gradle vs JDK

Este documento fornece informações sobre a compatibilidade entre as versões do Gradle e do JDK.

## Compatibilidade

| Versão Gradle | Versão JDK Suportada |
|---------------|----------------------|
| 7.3+          | 17                   |
