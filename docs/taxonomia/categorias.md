# Taxonomia de Categorias

Este documento define as categorias permitidas para a documentação do projeto Rebeca.

## Categorias

* **Funcionalidade**: Descreve funcionalidades específicas do jogo.
* **Desenvolvimento**: Detalhes sobre o processo de desenvolvimento.
* **Testes**: Informações sobre os testes realizados.
* **Requisitos**: Requisitos do sistema e do jogo.
* **Arquitetura**: Descrição da arquitetura do sistema.
* **Recursos**: Detalhes sobre os recursos utilizados (áudio, imagens, etc.).

## Padrões de Nomenclatura

* Utilizar letras minúsculas e separação por hífen para nomes de categorias.
* Evitar duplicatas e nomenclaturas inconsistentes.

## Guia de Uso

1. Verifique se a categoria desejada já existe na lista acima.
2. Se não existir, proponha uma nova categoria seguindo os padrões de nomenclatura.
3. Documente a categoria proposta com uma descrição clara.
