# Hospedagem da Documentação

A documentação do projeto Rebeca está hospedada no arquivo `README.md` dentro do repositório.

## Por que README.md?

- Acesso direto pelo repositório no GitHub.
- Facilidade de manutenção e atualização.

## Alternativas Consideradas

- Wiki do GitHub: Embora ofereça uma estrutura mais organizada, adicionaria complexidade desnecessária para o projeto.

## Checklist de Validação

- [x] Documentação acessível via README.md.
- [x] Conteúdo claro e conciso.
- [x] Links e referências atualizados.

Para modificar a hospedagem, execute o workflow "Definir Hospedagem da Documentação" disponível nas Actions do repositório.
