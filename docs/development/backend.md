# Instalação do ambiente de desenvolvimento Backend

## Pré-requisitos
- Node.js instalado
- npm ou yarn instalado

## Passos para instalação
1. Entre na pasta 
2. Execute  ou yarn install v1.22.19
[1/4] Resolving packages...
success Already up-to-date.
Done in 0.03s. para instalar as dependências
3. Configure as variáveis de ambiente necessárias (consulte a seção de configuração abaixo)
4. Execute  ou yarn run v1.22.19
info Visit https://yarnpkg.com/en/docs/cli/run for documentation about this command. para iniciar o servidor em modo de desenvolvimento

## Configuração
- Configure as variáveis de ambiente necessárias no arquivo  na raiz da pasta
