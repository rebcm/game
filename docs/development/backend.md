# Instalação do Backend para Desenvolvimento

## Pré-requisitos

- Node.js 18 ou superior
- npm ou yarn

## Passos para Instalação

1. Navegue até a pasta 
2. Execute  ou yarn install v1.22.19

3. Configure as variáveis de ambiente necessárias (consulte a seção de configuração abaixo)
4. Execute  ou yarn run v1.22.19
