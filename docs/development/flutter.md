# Instalação do ambiente de desenvolvimento Flutter

## Pré-requisitos
- Flutter instalado (versão especificada no )
- Dart instalado

## Passos para instalação
1. Clone o repositório do PassDriver
2. Execute  para instalar as dependências
3. Configure as variáveis de ambiente necessárias (consulte a seção de configuração abaixo)
4. Execute  para iniciar a aplicação em modo de desenvolvimento

## Configuração
- Configure as variáveis de ambiente necessárias no arquivo  na raiz do projeto

## Executando testes
- Execute  para rodar os testes unitários e de widget
