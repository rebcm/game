# Instalação

Este documento descreve os passos para instalar e configurar o ambiente de desenvolvimento do jogo.

## Pré-requisitos

* Flutter
* Android Studio (para desenvolvimento Android)
* Xcode (para desenvolvimento iOS)
* CocoaPods (para desenvolvimento iOS)

## Passos para instalação

1. Instale o Flutter seguindo as instruções em [flutter.dev](https://flutter.dev/docs/get-started/install).
2. Instale o Android Studio seguindo as instruções em [developer.android.com](https://developer.android.com/studio/install).
3. Instale o Xcode seguindo as instruções em [developer.apple.com](https://developer.apple.com/xcode/).
4. Instale o CocoaPods executando o comando `sudo gem install cocoapods` no terminal.

## Solução de problemas comuns

* Erro ao instalar CocoaPods: Verifique se o Ruby está instalado e atualizado.
* Erro ao executar o projeto no Android Studio: Verifique se o Flutter está configurado corretamente no Android Studio.
