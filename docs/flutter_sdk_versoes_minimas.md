# Versões Mínimas Suportadas pelo Flutter SDK

## Introdução

Este documento lista as versões mínimas de Android (API Level) e iOS suportadas pela versão atual do Flutter SDK utilizada no projeto game.

## Versões Mínimas

- **Flutter SDK:** $(grep "sdk: flutter" pubspec.yaml -A 1 | tail -n 1 | tr -d ' ')
- **Android:** API Level 21 (Android 5.0)
- **iOS:** iOS 12.0
