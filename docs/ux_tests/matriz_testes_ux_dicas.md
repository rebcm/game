# Matriz de Testes de UX para Dicas

## Introdução

Este documento define os cenários de teste para as dicas no jogo da Rebeca.

## Cenários de Teste

1. **Verificar se as dicas aparecem corretamente**
   - Pré-condição: O jogo está em execução.
   - Ação: O usuário clica no botão "Novo Bloco".
   - Resultado Esperado: A dica "Use o botão para criar um novo bloco" aparece.

2. **Verificar se as dicas são acessíveis**
   - Pré-condição: O jogo está em execução e a dica está visível.
   - Ação: O usuário navega até a dica.
   - Resultado Esperado: A dica é acessível e pode ser lida pelo leitor de tela.

3. **Verificar o fluxo de interação das dicas**
   - Pré-condição: O jogo está em execução e a primeira dica está visível.
   - Ação: O usuário clica em "Próximo".
   - Resultado Esperado: A próxima dica aparece.

## Conclusão

Esses cenários garantem que as dicas sejam apresentadas corretamente e sejam acessíveis aos usuários.
