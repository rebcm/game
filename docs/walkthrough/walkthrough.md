# Walkthrough

This document provides a step-by-step guide on how to use the game.

## Step 1: Starting the Game

1. Open the game on your device.
2. Click on the "Start" button to begin.

## Step 2: Building Blocks

1. Select a block from the inventory.
2. Click on the grid to place the block.

## Step 3: Exploring the Game World

1. Use the navigation controls to move around.
2. Explore the game world and discover new features.
