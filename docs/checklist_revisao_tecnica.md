# Checklist de Revisão Técnica

## Critérios Técnicos

- [ ] Nomenclatura de widgets
- [ ] Fluxos de navegação
- [ ] Termos de API

## Instruções

1. Verifique se a nomenclatura dos widgets está de acordo com o padrão estabelecido.
2. Verifique se os fluxos de navegação estão corretos e consistentes.
3. Verifique se os termos de API estão sendo usados corretamente.
