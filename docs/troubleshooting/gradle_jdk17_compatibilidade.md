# Compatibilidade do Gradle com JDK 17

Para garantir a compatibilidade com o JDK 17, é necessário utilizar o Gradle versão 7.3 ou superior.

## Como verificar a versão do Gradle

Execute o comando `gradle --version` no terminal para verificar a versão atual do Gradle.

## Solução para incompatibilidade

Se a versão do Gradle for inferior à 7.3, atualize o Gradle para uma versão compatível.
