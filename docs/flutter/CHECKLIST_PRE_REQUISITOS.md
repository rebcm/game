# Checklist de Pré-requisitos para Documentação do Flutter

## Versão do Flutter/Dart
- Versão do Flutter: 3.16.5 ou superior
- Versão do Dart: 3.2.3 ou superior

## SDKs Android/iOS
- Android SDK: API level 33 ou superior
- iOS SDK: 17.0 ou superior

## Variáveis de Ambiente
- : Chave de API para acesso ao backend
- : Ambiente de execução (dev, prod)

## Chaves de API Necessárias
- Chave de API do OpenStreetMap
- Chave de API do backend da PassDriver

## Passos de Validação do Build
1. Verificar se o Flutter e o Dart estão instalados e configurados corretamente.
2. Executar  para instalar as dependências.
3. Verificar se as variáveis de ambiente estão configuradas corretamente.
4. Executar  para validar o build e a execução do aplicativo.
