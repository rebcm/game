# Matriz de Resultados Esperados para Áudio

## Introdução

Este documento apresenta a matriz de resultados esperados para o comportamento do áudio no jogo em diferentes cenários e plataformas (Android e iOS).

## Cenários de Teste

1. **Modo Silencioso**
   - Descrição: Verificar o comportamento do áudio quando o dispositivo está configurado para modo silencioso.
   
2. **Alternância de Hardware**
   - Descrição: Testar o comportamento do áudio ao alternar entre diferentes saídas de hardware (por exemplo, fones de ouvido para alto-falantes).

## Matriz de Resultados Esperados

| Cenário          | Android                          | iOS                              |
|------------------|----------------------------------|----------------------------------|
| Modo Silencioso  | Áudio silenciado                 | Áudio silenciado                 |
| Alternância de Hardware | Áudio continua após troca       | Áudio continua após troca       |

## Considerações

- A matriz acima resume os comportamentos esperados para o áudio em diferentes cenários.
- Qualquer desvio desses resultados esperados deve ser investigado e corrigido.
