# Checklist de Aceitação para Documentação

## Critérios de Aceitação

- [ ] **Material**: O guia de construção especifica os materiais necessários.
- [ ] **Tempo**: O guia de construção estima o tempo necessário para a construção.
- [ ] **Dificuldade**: O guia de construção classifica a dificuldade da construção.

## Instruções

1. Verifique se o guia de construção segue o template definido.
2. Certifique-se de que todos os critérios de aceitação estão atendidos.
