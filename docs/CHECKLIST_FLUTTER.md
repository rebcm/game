# Checklist de Pré-requisitos Flutter

## Versões Necessárias

- Flutter SDK: >=3.0.0
- Dart SDK: >=2.17.0 <3.0.0

## Dependências de Sistema

- List of required system dependencies

## Como Validar

1. Execute `flutter --version` para verificar a versão do Flutter SDK.
2. Execute `dart --version` para verificar a versão do Dart SDK.
3. Verifique se as dependências de sistema necessárias estão instaladas.

## Checklist

- [ ] Flutter SDK instalado e versão >=3.0.0
- [ ] Dart SDK instalado e versão >=2.17.0 <3.0.0
- [ ] Dependências de sistema necessárias instaladas
