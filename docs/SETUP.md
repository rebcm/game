# Setup de Ambiente

## Versões Necessárias

- Flutter: >=3.0.0
- Dart: >=2.17.0 <3.0.0
- Java: (versão utilizada no projeto, verificar build-jdk17.yml)

## Passos para Configuração

1. Instalar Flutter na versão >=3.0.0
2. Instalar Dart na versão >=2.17.0 <3.0.0
3. Configurar Java de acordo com a versão utilizada no arquivo `.github/workflows/build-jdk17.yml`

## Verificação

Executar o script `.github/scripts/validate_flutter_version.sh` para verificar se as versões de Flutter e Dart estão corretas.
