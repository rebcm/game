# Ambiente de Desenvolvimento

Este documento fornece instruções para configurar o ambiente de desenvolvimento para o projeto Rebeca.

## Pré-requisitos

- Flutter SDK (versão >= 3.0.0)
- Dart SDK (versão >= 2.17.0)

## Configuração do Ambiente

### Windows

1. Instale o Flutter SDK seguindo as instruções em [Flutter Docs](https://docs.flutter.dev/get-started/install/windows).
2. Configure as variáveis de ambiente para o Flutter e o Dart.
3. Execute `flutter doctor` para verificar se o ambiente está configurado corretamente.

### macOS

1. Instale o Flutter SDK seguindo as instruções em [Flutter Docs](https://docs.flutter.dev/get-started/install/macos).
2. Configure as variáveis de ambiente para o Flutter e o Dart.
3. Execute `flutter doctor` para verificar se o ambiente está configurado corretamente.

### Linux

1. Instale o Flutter SDK seguindo as instruções em [Flutter Docs](https://docs.flutter.dev/get-started/install/linux).
2. Configure as variáveis de ambiente para o Flutter e o Dart.
3. Execute `flutter doctor` para verificar se o ambiente está configurado corretamente.

## Versões do Flutter SDK

O projeto é compatível com a versão >= 3.0.0 do Flutter SDK. É recomendável usar a versão mais recente para evitar problemas de compatibilidade.

## Solução de Problemas Comuns

- Se você encontrar problemas ao executar `flutter doctor`, verifique se as variáveis de ambiente estão configuradas corretamente.
- Se você tiver problemas com a compilação do projeto, certifique-se de que a versão do Flutter SDK é compatível.
