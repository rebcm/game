# Matriz de Critérios de Aceitação

## Introdução

Este documento define os critérios de aceitação para a feature de áudio do jogo.

## Critérios de Aceitação

1. **Escala de Volume**
   - O volume do jogo deve variar de 0 a 100%.
   - O volume deve ser ajustável pelo jogador.

2. **Comportamento do Botão Mute**
   - O botão mute deve silenciar todos os sons do jogo.
   - O botão mute deve ser toggle, permitindo que o som seja reativado.

3. **Persistência**
   - As configurações de volume e mute devem ser persistidas via SharedPreferences ou Hive.
   - As configurações devem ser carregadas ao iniciar o jogo.

## Conclusão

Estes critérios de aceitação garantem que a feature de áudio seja implementada corretamente e atenda às expectativas do jogador.
