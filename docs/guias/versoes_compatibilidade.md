# Matriz de Versões de Dependências

Este documento lista as versões compatíveis das dependências principais do projeto Rebeca Game, incluindo Flutter SDK, Dart e Java/Android Studio.

## Versões Compatíveis

| Dependência | Versão Mínima | Versão Máxima |
| --- | --- | --- |
| Flutter SDK | 3.0.0 | 3.3.10 |
| Dart | 2.17.0 | 2.19.6 |
| Java/Android Studio | 11 | 17 |

## Notas

- A versão mínima do Flutter SDK é 3.0.0 devido à dependência do `flutter_gl`.
- A versão máxima do Dart é limitada pela versão do Flutter SDK.
- A versão do Java/Android Studio é relevante para o desenvolvimento Android.

## Manutenção

Esta tabela deve ser atualizada sempre que houver uma mudança nas dependências principais do projeto.
