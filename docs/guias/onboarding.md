# Guia de Onboarding para Desenvolvedores

## Configuração Inicial

1. Clone o repositório `rebcm/game`.
2. Copie o arquivo `.env.example` para `.env`.
3. Preencha as variáveis de ambiente em `.env` com os valores necessários.

## Dependências

- Flutter
- Dart

## Executando o Projeto

1. Execute `flutter pub get` para instalar as dependências.
2. Execute `flutter run` para iniciar o aplicativo.

## Testes

- Execute `flutter test` para rodar os testes unitários.
