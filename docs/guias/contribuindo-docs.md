# Contribuindo para a Documentação

## Introdução

Este guia visa ajudar os contribuidores a entender como manter e atualizar a documentação do projeto.

## Boas Práticas

1. Sempre atualize a documentação relevante ao fazer mudanças no código.
2. Use markdown para formatação.
3. Siga o guia de estilo estabelecido.

## Checklist de Contribuição

1. [ ] Documentação atualizada
2. [ ] Checklists revisados
3. [ ] Glossário revisado

## Ação Corretiva

- Para qualquer item não conforme, revisar e atualizar a documentação.
