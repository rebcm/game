# Guia de Instalação

## Introdução

Este guia fornece as instruções necessárias para instalar e executar o projeto Rebeca.

## Pré-requisitos

* Flutter versão >= 3.0.0
* Dart SDK versão >= 2.12.0

## Configuração do Emulador

1. Instale o emulador Android Studio ou Xcode.
2. Configure o emulador para executar o projeto.

## Variáveis de Ambiente

* `FLUTTER_ROOT`: Caminho para o diretório raiz do Flutter.

## Instalação

1. Clone o repositório do projeto.
2. Execute `flutter pub get` para instalar as dependências.
3. Execute `flutter run` para executar o projeto.
