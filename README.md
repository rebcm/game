# Rebeca's Creative Block Game

## Instalação

To install and run this project, follow these steps:

1. Install Flutter and Dart.
2. Install Android Studio and configure the Android SDK.
3. Install CocoaPods for iOS development.

### Common Issues

* If you encounter issues with CocoaPods, try running `pod install` manually.
* If you encounter issues with Android Studio, ensure that the Android SDK is properly configured.
