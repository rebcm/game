# Critérios de Aceitação para Baselines de Performance

## Introdução

Este documento define os critérios de aceitação para os baselines de performance do jogo Rebeca.

## Baselines de Performance

Os seguintes baselines de performance devem ser atendidos:

* FPS médio: >= 55
* Número de jank frames: <= 10

## Verificação

A verificação dos baselines de performance será realizada automaticamente através do workflow de definição de baselines de performance.
