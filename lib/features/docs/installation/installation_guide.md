# Guia de Instalação do PassDriver

## Requisitos Obrigatórios

- Versão do Flutter: 3.10.6 ou superior
- Dart SDK: 3.0.6 ou superior
- Configuração de Emulador: Android Studio ou Xcode
- Variáveis de Ambiente: configuradas corretamente

## Passos para Instalação

1. Instale o Flutter e o Dart SDK
2. Configure o emulador de dispositivos
3. Configure as variáveis de ambiente
4. Execute o comando  para instalar as dependências
5. Execute o comando  para iniciar a aplicação

## Checklist de Verificação

- [ ] Versão do Flutter está correta
- [ ] Dart SDK está configurado corretamente
- [ ] Emulador está configurado e funcionando
- [ ] Variáveis de ambiente estão configuradas corretamente
