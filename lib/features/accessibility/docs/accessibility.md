## Opções de remapeamento de teclas e suporte a leitores de tela/controles adaptativos
O PassDriver suporta remapeamento de teclas e leitores de tela para uma experiência mais acessível.
### Remapeamento de teclas
Para remapear teclas, acesse as configurações do aplicativo e selecione 'Teclas'.
### Leitores de tela e controles adaptativos
O aplicativo é compatível com leitores de tela e controles adaptativos para facilitar a navegação.
