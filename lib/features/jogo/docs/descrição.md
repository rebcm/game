## Descrição da Experiência de Jogo
O jogo do PassDriver é uma experiência envolvente que simula a rotina de um motorista de ride-hailing no Brasil.
### Objetivos
- Completar corridas com sucesso
- Gerenciar tempo e recursos
### Características Principais
- Simulação realista de corridas
- Integração com mapas via OpenStreetMap
- Sistema de pontuação e ranking
