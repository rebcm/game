# Template de Dicas

## Estrutura Obrigatória

1. **Título**: Título da dica.
2. **Descrição**: Descrição detalhada da dica.
3. **Imagem**: Imagem ilustrativa da dica (opcional).
4. **Passos**: Passos para seguir a dica (opcional).

## Exemplo

### Título: Construindo uma Casa Simples
### Descrição: Uma casa simples para iniciantes.
### Imagem: casa_simples.png
### Passos:
1. Escolha um local plano.
2. Construa as fundações.
3. Adicione paredes e um teto.
