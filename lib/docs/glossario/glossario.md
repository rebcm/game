# Glossário de Termos Técnicos

Este glossário visa definir termos técnicos utilizados no projeto para garantir consistência.

## Termos

- **Voxel**: Unidade cúbica que compõe o mundo do jogo.
- **Bioma**: Região com características específicas (apesar de não termos múltiplos biomas, é importante definir).
- **Rebeca**: Personagem principal do jogo.

## Uso

Este glossário deve ser consultado para garantir que os termos sejam usados consistentemente em todas as partes do projeto, incluindo código, documentação e comunicação.
